## 🖥️ Projeto

O Projeto é uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnólogias

Esse projeto foi desenvolvido durante o NLW da Rockset con as seguintes tecnologias:

- HTML
- CSS
- Git e Github
